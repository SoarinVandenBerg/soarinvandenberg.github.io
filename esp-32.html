ESP-32 Project



This is where ill put all the info for my ESP-32 project. I don't think anyone will read this, but im just going to use this as a spot to keep all the info so I don't lose it.

This project consists of ~25 games, a bunch of basic apps, and some other useful stuff. It has the same theme as this website, and it's got a bunch of other features such as a stats logger, settings page, multiplayer and singleplayer on most games, and some other stuff. I originally made the globe pixel art animation for the idle screen for the device, but am just using the freeze frame, but theres an option to revert in settings.

About the making of this, as explained in my "thoughts about AI page", I used AI, specifically claude code (Opus 4.7) heavily on this project. It did most of the coding, but I reviewed it all and there shouldnt be too many bugs. Without using it, I don't think this project would have been possible, or it would have just been smaller and/or worse.

Before even considering doing this, read all of the instructions below.

Step 1: Buy the device. It costs around 50$ canadian depending on where you buy it, but if you get it on the waveshare website, it comes with a battery. Anywhere else i've found you have to buy it seperately which is usually around 10$ canadian. (here's the link to the site to buy it)

Step 2: Install Thronny and also download all the code (below) for the project. Then put the bootloader file onto it using a python script (I completely forget how to do this part, and im too lazy rn to re-learn so I asked Claude to make a file of instructions, (its in the project files) just contact me if you need help). Here's the repo with the code.

Step 3: Connect the ESP-32 via USB-C to whatever computer you're using for this. To make it connect, just look in the setup instructions file in the Github repository. Then, put the rest of the files in thronny (this should be pretty easy to figure out). It does take a while to transfer the files, but it shouldnt be too long.

Step 4: Press CTRL+D to restart the device and boot it up. CTRL+C reboots it. Make sure to look up a tutorial for how to use thronny first. Once you unplug it from your computer, press and hold both buttons for ~10s until the screen flashes, then hold the PWR button until you see the idle screen.

Now if everything has worked so far (very unlikely), you should see an idle screen (a globe) and if you tap the screen, you should see a list of apps! The touch controls are pretty finnicky, but everything should be pretty intuitive. If you need any help, feel free to contact me!

That should be it! To do this project, you do need to be familliar with thronny or coding, and you will most likely encounter problems with stuff as ive only done this twice and each time different problems occured. If I havent said it already, DO THIS AT YOUR OWN RISK!! There's probably a buncha stuff I left out here, so ask if needed.
