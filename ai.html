My Thoughts on AI


I'm just going to use this page as a place to talk about my thoughts on AI.

My stance on AI is that it depends on what you're doing and why. For
example, my goal for this website is to not use AI at all. For my other projects
like the ESP-32, AI did most of the code, because I couldn't achieve my goals
without it. I think using AI is a choice that you can make, but you should make
it clear that you used it. If your goal is to learn how to code, or you don't know
how yet, then you shouldn't use it, but if you know how, and you're using it as
a tool, that works.

I'm not even going to try to argue about the environmental impacts, I
don't think I'm informed enough about the subject. I’m talking about using AI
for coding here, but if you're using it for anything else, you should double-
check the info it gives you.
